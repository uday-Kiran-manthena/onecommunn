## Setup and Running

Open Terminal/Command Prompt:

1. **Clone the repository :**

   ```bash
   https://github.com/uday-Kiran-manthena/onecommunn.git

   ```

2. **Navigate into the project directory :**

```bash
cd onecommunn
```

3. **Install Dependencies :**

```bash
npm install
```

4. **Start the App :**

```bash
npm start
```
